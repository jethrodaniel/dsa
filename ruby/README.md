# DSA

Data structures and algorithms practice.

## Setup

Install ruby:

```
$ asdf install
$ ruby -v
ruby 3.4.1 (2024-12-25 revision 48d4efcb85) +PRISM [x86_64-linux]
```

Install this gem

```
git clone https://github.com/jethrodaniel/dsa
cd dsa/ruby

bundle
```

## Run tests

```
bin/rake
```

## View documentation

```
bin/rake doc
```

Then visit http://localhost:8808/
